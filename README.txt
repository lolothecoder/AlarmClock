Nous avons choisi de créer un nouveau circuit pour chaque module ensuite l'importer dans le MAINCIRCUIT pour
plus d'organisation (avoir les modes comme des librairies). Le projet final se trouve dans MAINCIRCUIT.Voici
une brève explication de chaque circuit:

- Adding/AddingMSG : simples circuits additionneur

- Alphabet/NumberAffichage : convertit les lettres et les chiffres en code lisible pour un 7segment

- LEDController : l'effet LED dans le mode "MUSIC"

- time_set_bus : le code de l'horloge et le code pour modifier l'heure et la date

- setAlarmTimer : un setter pour l'alarme, le timer et la calculatrice

- MSG, Alarm, MusiqueUnggatedAttempt2, Timer, chrono_time, zone_time_final, Alarm:
les codes pour les différents modes.
